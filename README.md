# LearnRust

This repository contains my notes, examples, and command history while working
through [The Rust Programming Language](https://doc.rust-lang.org/book/).

The goal of this project is to keep everything I learn from the book in one
place, including:

- Chapter-wise notes and summaries
- Rust source code written while following the examples
- Terminal commands used during setup, compilation, testing, and experiments
- Small exercises and practice programs inspired by the book

This is a learning repository, so the contents may evolve as I revisit chapters,
refactor examples, or add more detailed explanations.
